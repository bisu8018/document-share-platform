# blue-bottle

## Truffle
> npm install -g truffle

## Ganache-CLI
> npm install -g ganache-cli

## Create-React-App
> npm install -g create-react-app

## Running Ganache
> ganache-cli -b 3

## SmartContract Compile
> truffle compile

## SmartContract Migrate
> truffle migrate

## Choose Network
> truffle migrate --network rinkeby

## nodejs start
>> cd client
npm start
